# 🚐 TravelTrucks

TravelTrucks is a camper rental web application built with **Next.js**, **TypeScript**, and **React Query**.

The application allows users to browse available campers, filter them by different parameters, view detailed information, read reviews, and submit a booking request.

## ✨ Features

- Home page with Hero section
- Camper catalog
- Filtering by:
  - Location
  - Camper form
  - Engine type
  - Transmission
- Pagination (Load More)
- Camper details page
- Image gallery
- Reviews
- Booking form
- Loading states
- Toast notifications
- "No campers found" state
- Responsive layout

---

## 🛠 Tech Stack

- Next.js 16
- TypeScript
- React
- React Query (TanStack Query)
- CSS Modules
- Axios
- React Hot Toast
- React Icons

---

## 🚀 Getting Started

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Start production server

```bash
npm run start
```
